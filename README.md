## 🛠️ Module Descriptions & Breakdown

### 🔹 Task 1: Online Railway Reservation System
*   **File Name:** `Athilakshmi_Task1.java`
*   **Overview:** A console-driven railway routing application simulating customer database setups, unique record mappings, and session tracking.
*   **Key Features:**
    *   Dynamic dynamic client credentials registration and verification profiles.
    *   Dynamic entry reservation collection storing passenger name, train number, travel classification, and location.
    *   Automated lookups matching dynamic keys (e.g., `12601` -> Chennai Mail, `12673` -> Cheran Express).
    *   Auto-incremented unique PNR system tracking references beginning from `55001`.
    *   Strict cancellation processing removing entries securely through targeted PNR indices.

### 🔹 Task 2: Number Guessing Game (Swing GUI)
*   **File Name:** `Athilakshmi_Task2.java`
*   **Overview:** A graphical Java Swing UI guessing arcade application requiring calculations against randomized numerical generations.
*   **Key Features:**
    *   GUI prompt communication via functional interactive `JOptionPane` dialog panels.
    *   Pseudo-random numeric generation handling structural fields scaled from 1 to 100.
    *   Iterative score limits restricted to a maximum threshold of 5 evaluation attempts per round.
    *   Dynamic relative clue pointers signaling directions ("Go Higher ↑" or "Go Lower ↓").
    *   Adaptive point metric engines multiplying current attempt counts by bonus parameters.

### 🔹 Task 3: Automated Teller Machine (ATM) Interface
*   **File Name:** `Athilakshmi_Task3.java`
*   **Overview:** A secure Object-Oriented application displaying system architectures managing distributed financial balance properties.
*   **Key Features:**
    *   Decoupled multi-class logic framework (`LogManager`, `UserWallet`, `CredentialsRegistry`, `ServiceDashboard`).
    *   Historical logging modules processing running transaction timestamp strings.
    *   Financial validation managing credit addition and active constraint boundaries (Daily Max: INR 20,000).
    *   Secure structural verification engines for registration profile credentials and active PIN alterations.
    *   Console dashboard executing withdrawals, deposits, history overview dumps, and remittances.

### 🔹 Task 4: Online Examination Engine (Multi-Threaded)
*   **File Name:** `Athilakshmi_Task4.java`
*   **Overview:** An interactive testing application showcasing multi-threaded system setups managing strict evaluation timelines.
*   **Key Features:**
    *   Concurrent `Thread` countdown timers configuring strict assessment timelines (120-second threshold limits).
    *   Asynchronous terminal input validation stopping question workflows immediately when clocks timeout.
    *   Secure dynamic terminal authentication panels managing profile creations and active password rewrites.
    *   Dynamic arrays configuring technical question trivia blocks (Java Architecture, LIFO, Framework types).
    *   Automated diagnostic metrics calculators rendering exact performance percentages instantly upon exit.

### 🔹 Task 5: Digital Library Management Network
*   **File Name:** `Athilakshmi_Task5.java`
*   **Overview:** A dual-portal library automation program managing centralized inventory configurations, checkouts, and fine calculation analytics.
*   **Key Features:**
    *   Separated access workflows managing targeted modules (**Management Desk** & **Patron Interface**).
    *   Dynamic collection modification engines supporting live text categorization and indexing drops.
    *   Automated deadline delay metrics calculating flat penalty costs (INR 5/day overdue logic).
    *   Priority booking holding reservation registers when materials are currently loaned out.
    *   Administrative analytics pipelines tracking total active inventory sizes and collection revenue.

## 🚀 Execution Instructions

To execute any specific task locally on your device, use the following terminal workflow:

1. **Verify Environment:** Ensure Java Development Kit (JDK) is successfully configured on your local system path.
2. **Compile Project:** Execute the standard compiler pointing directly to the target task file name:
   ```bash
   javac Athilakshmi_Task1.java
   ```
3. **Run Application:** Launch the compiled platform module using the execution string:
   ```bash
   java Athilakshmi_Task1
   ```
